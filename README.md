# Connect 4 Circuit

Authors:
* Sebastian Häni
* Raphael Laubscher
* Lukas Näf

## Purpose

The circuit represents a 2-player connect 4 game without win detection. 
Each player has 7 buttons to select in which column he wants to drop a coin.
The Start button resets the game.

## Installation

* Install Logisim: http://ozark.hendrix.edu/~burch/logisim/
* Make sure the circ file is on your computer an open connect4.circ with Logisim
* Enable Simulation in Logisim
* Press the Start button
